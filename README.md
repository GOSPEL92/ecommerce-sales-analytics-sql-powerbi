
# E-Commerce Sales Analysis (SQL + Power BI)
SQL queries and Power BI dashboard analyzing e‑commerce sales, customer behavior, and revenue trends.
##  Overview
This project analyzes an e-commerce dataset using SQL queries and Power BI dashboards.  
Key insights include:
- Top 10 products by sales and revenue
- Top 10 countries by sales
- Monthly sales trend
- Customer segmentation (RFM analysis)

## Tools Used
- SQL Server
- Power BI
- Git
- Github

## Repository Structure
- `sql_scripts/` → SQL queries with comments
- `data/` → dataset (CSV)
- `powerbi_dashboard/` → Power BI report file
- `images/` → dashboard screenshots

##  Insights
- Certain products dominate revenue despite lower sales volume.
- The UK is the largest market, followed by Germany and France.
- Seasonal spikes in sales around November–December.
- RFM segmentation highlights loyal high-value customers and at-risk segments.
